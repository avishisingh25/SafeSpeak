# SafeSpeak

SafeSpeak is an AI-powered Social Media Hate Speech & Gender-Based Abuse Detection System.

## Problem Statement

Women frequently face online harassment, hate speech, and abusive comments across social media platforms. Manual moderation is slow and ineffective at scale.

## Solution

SafeSpeak uses Natural Language Processing (NLP) and transformer-based AI models to automatically identify toxic, abusive, and gender-based harmful content in real time.

## Features

* Hate Speech Detection
* Gender-Based Abuse Detection
* Toxicity Classification
* Confidence Score Generation
* Real-Time Analysis
* User-Friendly Interface

## Tech Stack
